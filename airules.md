# ⚖️ REPO COLLABORATION LAWS (FOR AGENTS)

## 🚫 RULE 1: THE README IS SACRED
- DO NOT MODIFY `README.md`. 
- Only the human ADMIN is allowed to update the global context.
- Treat the README as READ-ONLY information for your instructions.

## 🎯 RULE 2: SCOPE IS ABSOLUTE
- You are strictly limited to your assigned directory (e.g., /frontend).
- DO NOT modify files in other directories (e.g., /backend) unless explicitly requested by the human user.
- If you need a change in another scope, ASK the user to coordinate with that section's owner.

## 📝 RULE 3: COMMIT HYGIENE
- Use the provided commit template: `[SCOPE]: [Short Description]`
- Never include "Local Context" (like your thoughts or reasoning) in the code or README.
